# Getting Started

Welcome to the **DevOps Blog** — a living knowledge base for Cloud and Infrastructure topics.

## What You'll Find Here

This site is organized into focused sections:

| Section | What's Covered |
|---------|----------------|
| **Linux** | Commands, shell scripting, administration |
| **Networking** | Core concepts, troubleshooting, DNS, firewalls |
| **Google Cloud** | GCP services, GKE, IAM, Terraform on GCP |
| **Kubernetes** | Deployments, services, troubleshooting |
| **Terraform** | Infrastructure as Code, modules, best practices |

## Who Is This For?

- Cloud Engineers working with GCP and Kubernetes
- DevOps practitioners building CI/CD and IaC pipelines
- Students preparing for certifications (GCP Associate, CKA, etc.)
- Anyone curious about the mechanics of modern infrastructure

## How to Navigate

- Use the **Docs** sidebar to browse topics by category
- Visit the **Blog** for deep dives, tutorials, and exam tips
- Use the **Search** bar to find keywords across all pages

## Quick Start

```bash
# Clone the site repository
git clone https://github.com/souravlobra/devops-blog.git
cd devops-blog

# Install dependencies and start locally
npm install
npm run start
```

Happy learning!
