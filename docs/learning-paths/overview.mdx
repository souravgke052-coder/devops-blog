# Learning Paths Overview

Welcome to our structured learning paths. Each path provides a clear, progressive curriculum from beginner to advanced level, with hands-on labs, real projects, and certification alignment.

## Available Paths

### By Experience Level
- **[Beginner Path](/learning-paths/beginner)** — Start your DevOps journey with foundational skills
- **[Intermediate Path](/learning-paths/intermediate)** — Build on fundamentals with production-ready skills
- **[Advanced Path](/learning-paths/advanced)** — Master complex systems and architectural decisions

### By Role
- **[Role-Based Paths](/learning-paths/role-based)** — Specialized curricula for specific careers:
  - Cloud Engineer / Platform Engineer
  - SRE / DevOps Engineer
  - Security Engineer
  - Data Engineer (Infrastructure Focus)

## How Learning Paths Work

Each learning path follows the same structure:

| Component | Description |
|-----------|-------------|
| **Prerequisites** | What you need before starting |
| **Phases** | Progressive skill building (4 phases per path) |
| **Labs** | Hands-on exercises reinforcing each concept |
| **Projects** | Real-world portfolio projects |
| **Certifications** | Aligned certification preparation |

## Path Structure

### Phase 1: Foundation
- Core concepts and terminology
- Basic tool introduction
- Simple hands-on exercises
- Validation checkpoints

### Phase 2: Core Skills
- Technology-specific deep dives
- Multi-tool workflows
- Intermediate labs
- Project work

### Phase 3: Advanced Topics
- Architectural patterns
- Complex integrations
- Advanced labs and challenges
- Optimization techniques

### Phase 4: Production Readiness
- Production deployment patterns
- Security hardening
- Monitoring and alerting
- Incident response
- Portfolio project completion

## Time Estimates

Each path requires a time investment based on your learning pace:

| Intensity | Hours/Week | Path Duration |
|-----------|------------|---------------|
| Part-time | 5-10 hours | 4-6 months |
| Full-time | 20-30 hours | 1-2 months |
| Accelerated | 40+ hours | 3-4 weeks |

## Choosing Your Path

### Start Here If You Are:
- **New to DevOps**: Begin with [Beginner Path](/learning-paths/beginner)
- **Transitioning from another role**: Start with [Beginner Path](/learning-paths/beginner), then move to [Role-Based](/learning-paths/role-based)
- **Experienced but unstructured**: Take the [Intermediate Path](/learning-paths/intermediate) for gaps assessment
- **Senior engineer building expertise**: Go directly to [Advanced Path](/learning-paths/advanced)

### Path Assessment

Not sure where to start? Take this quick assessment:

1. Can you navigate Linux and write basic shell scripts?
2. Do you understand containers and container orchestration?
3. Have you deployed infrastructure as code before?
4. Can you design and implement CI/CD pipelines?
5. Have you worked with cloud providers in production?

**If you answered "no" to 3+ questions**: Start with [Beginner](/learning-paths/beginner)

**If you answered "no" to 1-2 questions**: Start with [Intermediate](/learning-paths/intermediate)

**If you answered "yes" to all**: Start with [Advanced](/learning-paths/advanced)

## Success Metrics

Track your progress through each path:

1. **Lab Completion** — All hands-on labs completed and validated
2. **Project Portfolio** — Portfolio projects demonstrating skills
3. **Certification** — Relevant certifications achieved
4. **Peer Teaching** — Ability to explain concepts to others
5. **Production Experience** — Real production systems managed
